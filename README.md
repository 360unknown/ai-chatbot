## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://github.com/Ovobot/pxt-xtronpro](https://github.com/Ovobot/pxt-xtronpro)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/360unknown/ai-chatbot** and import

## Edit this project ![Build status badge](https://github.com/360unknown/ai-chatbot/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* open [https://github.com/Ovobot/pxt-xtronpro](https://github.com/Ovobot/pxt-xtronpro)
* click on **Import** then click on **Import URL**
* paste **https://github.com/360unknown/ai-chatbot** and click import
